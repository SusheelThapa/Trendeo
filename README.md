## Installation

### Prerequisites

Ensure you have `Node.js` and `npm` or `yarn` installed on your machine.

### Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/SusheelThapa/Trendeo.git
   cd Trendeo
   ```

2. Installation of node packages

   ```bash
    yarn
   ```

3. Running the project:

   ```bash
   yarn run dev
   ```
